This analysis has several limitations. The score uses predefined weights that favor goals and assists, so offensive players naturally receive an advantage. The dataset does not include minutes played, player positions, opponent strength, injuries, turnovers committed, or game situations. Therefore, the score should support coaching decisions, not replace player evaluation and game-film analysis.
